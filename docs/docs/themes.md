Gatsby 主題將一個網站的默認配置（共享的功能，數據源，設計）簡化成一個安裝包。

這意味著你的項目的配置和功能不是被編寫進去的，而是通過安裝一個版本化的，集中管理的依賴。你可以無縫地更新一個主題，與其他人一起創作一個主題，甚至還可以將一個兼容的主題替換成另外一個。
